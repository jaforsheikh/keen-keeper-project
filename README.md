## keen-keeper-project

KeenKeeper is a modern web application designed to help you maintain meaningful relationships by tracking your interactions with friends and ensuring you stay connected.

 ## Overview

KeenKeeper acts as your **personal relationship manager**.  
It helps you:
Track how long it’s been since you contacted someone  
Get reminders when a connection needs attention  
Log interactions like calls, texts, and video chats  

## Technologies Used

Next.js 14 (App Router)
React.js
Tailwind CSS
DaisyUI
Recharts (for analytics)
React Toastify
LocalStorage 

## Key Features

## 1. Smart Dashboard
View all your friends in a clean card layout  
Track status: *On Track, Almost Due, Overdue*  
Real-time stats (Total Friends, Need Attention, etc.)

## 2. Dynamic Friend Profile
Click any friend to view detailed profile  
Shows:
Days since contact  
Goal (days)  
Next due date  
Quick actions:
Call 📞  
Text 💬  
Video 🎥  
Each action saves to timeline + shows toast notification

### 3. Timeline & Analytics
Timeline page logs all interactions dynamically  
Filter by:
Call
  - Text
  - Video  
- Stats page shows:
  - Pie chart of interaction types  
  - Visual insights using Recharts
